# PyNet-Scanner
Desarrollo de un escáner de puertos de alto rendimiento basado en Sockets. Implementa de forma nativa el Three-Way Handshake de TCP para identificar puertos abiertos en el rango completo (1-65535). Optimizado mediante programación concurrente (Multithreading) con 500 workers para maximizar la velocidad de descubrimiento sin dependencias externas.
